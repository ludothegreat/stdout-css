DESIGN PRINCIPLES

Every pixel serves a purpose. No marketing fluff, no decorative elements.

SPEED

Instant Feedback

Natural deceleration animations (ease-out-quad, ease-out-expo). No bounces, no elastic effects. Just smooth, purposeful motion.

CLARITY

At A Glance

Information-dense layouts with clear visual hierarchy. Status indicators, badges, and color-coded elements make state obvious.

TRUST

Precise Feedback

Technical precision over hand-holding. Accurate terminology, exposed details, and clear success/error states.

COLOR

No Blues

OKLCH perceptually uniform colors. Greens for primary actions, burnt orange for secondary. Blue is explicitly excluded.

DARK MODE

Low Light First

Designed for developers working in low-light environments. High contrast, reduced eye strain. Light mode supported.

TYPOGRAPHY

Mono For Data

Commands, hashes, timestamps, code—all monospace. Sans-serif for prose. Clear hierarchy, brutal honesty.

COMPONENTS

Buttons, forms, tables, badges, modals, toasts, tabs, accordions, tooltips, command palettes, code boxes, loading states, and more.
